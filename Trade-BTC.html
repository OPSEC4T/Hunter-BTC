Domande Frequenti

È sicuro collegare le mie API Binance?

Assolutamente. Le API keys rimangono nel file .env sul TUO server locale. Hunter-BTC non invia mai le chiavi all'esterno.

Quanto capitale serve per iniziare?

100 EUR sono il minimo consigliato. Il capitale iniziale è configurabile tramite telegram con un messaggio al bot.

Serve un VPS o posso usare il mio PC?

Puoi usare qualsiasi macchina che supporti Docker (Raspberry Pi, vecchio Laptop, VPS da 5$/mese). Deve rimanere accesa H24.
